# SkillSurge.io

A simple responsive landing page built with HTML, CSS, and JavaScript.

## Overview

SkillSurge.io is a single-page website for a cohort-based learning platform. It includes a navigation-based layout, responsive styling, modern section design, and interactive elements built with vanilla JavaScript.

## Project Includes

1. Page with navigation
2. Responsive styles with CSS
3. Flexbox and CSS Grid
4. Deployment-ready website structure

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- [Google Fonts](https://fonts.google.com/)
- [Font Awesome](https://fontawesome.com/)

## Project Structure

```text
SkillSurge-io/

```

## Sections

- Home
- Platform
- Tracks
- Features
- FAQ
- Join Now / Contact
- Footer

## Interactivity

The JavaScript layer currently handles:

- Mobile menu open/close behavior
- Active navigation link highlighting on scroll
- FAQ accordion expand/collapse behavior
- Contact form submit feedback with temporary success button state

## Getting Started

### Prerequisites

You only need a modern web browser.

### Run Locally

1. Clone or download this repository.
2. Open the project folder.
3. Launch `index.html` in your browser.

You can also use a lightweight local server if you prefer, such as VS Code Live Server.

## Layout and Styling

- Built with responsive CSS
- Uses Flexbox for alignment and component layout
- Uses CSS Grid for section and card layouts
- Adapts across desktop, tablet, and mobile screens

## External Dependencies

This project loads the following assets from CDNs:

- Google Fonts
- Font Awesome 6.5.0

An internet connection is required for those external assets to load unless they are hosted locally.

## Deployment

This project is ready to be deployed as a static website using platforms like GitHub Pages, Netlify, or Vercel.
